# 📚 HomaStationDocs - База знаний сервера ClownHoma
---
**Добро пожаловать!** Это место, где мы собираем опыт, решения и грабли, на которые наступили, чтобы не наступать дважды.

---
## 🎯 Для чего это?
- Быстро находить ответы на частые вопросы
- Не держать знания в головах и чатах
- Не изобретать велосипед каждый раз
- Видеть каких ответов НЕ хватает

## 🔍 Как искать?
**Самый быстрый способ:** В корневой папке нажать *Ctrl+F* и ввести ключевое слово.
**Интуитивный способ:** Открыть [https://repentordie.github.io/HomaStationDocs/](https://repentordie.github.io/HomaStationDocs/), просмотреть оглавление по нужной теме и перейти на нужный гайд.

## ❗️ Что делать, если ответ НЕ НАШЕЛСЯ?
**Грустно, но это не провал, а ценный сигнал для нашей команды!!!**

У тебя есть два варианта:
### ВАРИАНТ А: У тебя ЕСТЬ решение
Перейди во вкладку **Issues** → **New Issue** → выбери шаблон **"📚Предложить гайд"**.
Заполни поля - я оформлю твой опыт в гайд.

### ВАРИАНТ Б: У тебя НЕТ решения(в тупике)
Перейди во вкладку **Issues** → **New Issue** → выбери шаблон **"❓Проблема без решения"**.
Заполни поля - это Issue становится **задачей команды**: кто-то решит проблему, ты получишь ответ, а я превращу это в новый гайд.

**Почему это важно:** Если ты не нашел ответ - значит, в базе дыра. Закрыв её мы сохраняем лишние два часа следующему кто столкнётся с такой проблемой

## 📁 Структура
*Пока курим... всё враки...*

| Папка           | Что там                                                        |
| --------------- | -------------------------------------------------------------- |
| ~~guides/~~     | ~~Готовые, отредактированные гайды, рассортированы по темам.~~ |
| ~~inbox/~~      | ~~Сырые черновики~~                                            |
| ~~issues/~~     | ~~Описание проблем без решения (переехавшие из Issues)~~       |
| ~~meta/templates/~~ | ~~Шаблоны для оформления~~                                         |
| ~~meta/~~           | ~~Внутренние заметки по процессам KM, вложения~~                   |
- Некоторые гайды содержат ==подобное форматирование==, что означает что информация требует пересмотра. В случае если целый гайд требует актуализации, в начале файла будет соответствующая надпись.
- ~~Подобное форматирование~~, означает что информация не актуальна т.к. ещё не реализована, но в ближайшем будущем это поменяется.

##  Как помочь, если ты решил чужую проблему?
Видишь в Issues открытый вопрос "[ПРОБЛЕМА]"?
- Напиши в комментариях, как ты решил.
- Я перенесу это в гайд и закрою Issue.
